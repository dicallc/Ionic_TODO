## 使用 ##

1.npm i 安装依赖

2.ionic serve或者运行ionic run android

    保证你已经安装好了ionic环境,如果没有 npm install -g ionic


## 你能学到什么 ##

1.angular 4和ionic 3的新写法

2.ionic导航跳转以及传值

3.使用Storage进行持久化
